# Adaptive Learning Content System

This repository contains the full source code for the Adaptive Learning System, featuring a **FastAPI** backend and a **Vanilla JS** frontend.

## Structure

- **`/backend`**: Python FastAPI application with Supabase & Gemini integration.
- **`/frontend`**: HTML/CSS/JS dashboard for user interaction.

## Quick Start

### Backend
1. `cd backend`
2. `python -m ven venv` & `source venv/bin/activate`
3. `pip install -r requirements.txt`
4. Set up `.env` (see `.env.example`).
5. `uvicorn main:app --reload`

### Frontend
1. Open `frontend/index.html` in your browser (or use Live Server).
